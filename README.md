# Cliente/Servidor TCP-IP

## Instalação

Baixe o repositório utilizando:
```shell
$ git clone https://github.com/networkfga/tcp-ip.git
```

Dentro da pasta execute:
```
$ make
```

## Execução

### Cliente
```
$ make exec_client
```

### Servidor
```
$ make exec_server port=PORTA
```
Exemplo:
```
$ make exec_server port=2152
```
